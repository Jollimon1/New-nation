## Priority Issues

**[P1] Body text contrast fails WCAG AA** — `text-zinc-500` on white = ~4.48:1, below 4.5:1 minimum. Affects hero paragraph, ecosystem descriptions, FAQ answers. Fix: use `text-zinc-600`.

**[P1] No prefers-reduced-motion** — All animations (marquee, bounce-slow, scanline, framer-motion reveals) run unconditionally. Fix: add reduced-motion media query to all keyframes; use useReducedMotion() hook in components.

**[P2] Side-stripe border on hero paragraph** — `border-l-2 border-[#FF6B00]` is an absolute ban. Fix: remove entirely; let the paragraph breathe.

**[P2] Identical card grid in Ecosystem** — 4 cards, same size, same structure. Fix: restructure as a tiered flow showing escalating capability.

**[P2] Navigation breaks browser back button** — `setCurrentView` state switching has no URL changes. Fix: implement React Router.
